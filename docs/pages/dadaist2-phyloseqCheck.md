## dadaist2-phyloseqCheck
**dadaist2-phyloseqCheck** - Check PhyloSeq object from the command line

## Author
Andrea Telatin and Rebecca Ansorge

## Parameters
- _-i_, _--input_ FILE

    Input file in in PhyloSeq object (R Object)

- _-j_, _--json_

    Output in JSON format
